📊 Sales Data Analysis (EDA Project)

🔍 Problem Statement:

Analyze customer order data to understand spending patterns and delivery performance.

📁 Dataset:

200 orders
Features: city, category, order value, delivery days, rating

🧹 Data Cleaning:

Handled missing values
Treated outliers

📊 Analysis Performed:

Distribution of order values
Delivery time analysis
Correlation analysis
City-wise and category-wise sales

🔥 Key Insights:

Most customers spend low amounts
Few orders are very expensive (outliers)
Delivery time is consistent
No strong relationship between variables

🛠 Tools Used:

Python
Pandas
NumPy
Matplotlib
Seaborn
